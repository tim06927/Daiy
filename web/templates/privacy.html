What We Store

This alpha demo stores the following data for debugging and improvement purposes:

  • User inputs: text descriptions, uploaded photos, model/effort selections, clarification choices and answers
  • Generated content: LLM prompts and responses
  • Technical logs: error logs and runtime logs

We apply automatic redaction to remove obvious personal patterns (email addresses, phone numbers) before storing text data.

What We Don't Store

  • IP addresses
  • Stable user identifiers or tracking cookies
  • Personal data beyond what you explicitly enter

Where Data Is Stored

Data is stored in an SQLite database on the Render server hosting this application.

Purpose

The stored data is used solely for debugging issues and improving this alpha demo. It is not used for marketing, profiling, or shared with third parties beyond what is described below.

LLM Processing

Your prompts are sent to OpenAI to generate responses. OpenAI's data collection options are disabled for this service, meaning OpenAI does not use your inputs for training their models.

Data Retention

All stored data is automatically deleted after 90 days.

Deletion Requests

To request deletion of your data before the 90-day retention period expires, please email hello@daiy.de.
